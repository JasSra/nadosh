Install Edge Agent

� Quick Install

One-liner installation (pipe to shell):

PowerShell (Windows/Linux):
iwr /edge/install.ps1 | iex
Bash (Linux):
curl -sSL /edge/install.sh | bash

The script will prompt for API key and site ID.

🔑 Non-Interactive Install

Use environment variables to skip prompts:

# PowerShell
$env:NADOSH_API_KEY="your-api-key"
$env:NADOSH_SITE_ID="aws-east"
iwr /edge/install.ps1 | iex

# Bash
NADOSH_API_KEY="your-api-key" \
NADOSH_SITE_ID="aws-east" \
curl -sSL /edge/install.sh | bash

📥 Manual Download

Download and run locally:

PowerShell: /edge/install.ps1
Bash: /edge/install.sh

✅ Verify Installation

The agent will automatically:

  • Enroll with the mothership control plane
  • Send heartbeats every 60 seconds
  • Poll for authorized tasks every 30 seconds
  • Execute tasks matching its capabilities

Check enrollment status in the tab.

📋 API Endpoints

Enrollment: POST /v1/edge-agents/enroll
Heartbeat: POST /v1/edge-agents/heartbeat
Get Tasks: GET /v1/edge-agents/{agentId}/tasks
Claim Task: POST /v1/edge-agents/{agentId}/tasks/{taskId}/claim
